#### Requirements ###
-------------------------------------------------------------------------------------------------------------------------
--> Python ^3.7
--> The project was develop in VCode. Inside the project you will find a venv example that you can use. 
    - If not, user can create their own venv using dependencies defined in 'Requirements.txt' (Besides core libraries I used pytest and coverage for testing purposes)
        -> Once venv is activated --> executes "pip3 install -r requirements.txt"
    - Project was tested with terminal and it works perfectly.

--> Set variables "file_path" and "n_limit" in main.py to run the process
    - file_path (string) --> path + file to be processed. I have created a folder called input_data to deposit files
    - n_limit (int) --> names to be processed
-------------------------------------------------------------------------------------------------------------------------
### Project structure ###
-------------------------------------------------------------------------------------------------------------------------
-> You can find comments related to the code inside each function and a general description in main.py
root
    [Folder]    input_data (to deposit files to process)
    [Folder]    results (here you will find results generated by the program)
    [Folder]    src 
                [Folder] core (modules)
                        constants.py --> constants
                        logs.py --> log module
                        html_generator.py --> report module
                        util.py --> methods to execute all the tasks
                [Folder] exceptions (exceptions modules)
                        util_ex.py --> exceptions for util module
    [Folder]    tests (unit tests by module)
                *main.py (module to executes)
                readme.txt (this file)
                requirements.txt (dependencies)

-------------------------------------------------------------------------------------------------------------------------
### How to run ###
-------------------------------------------------------------------------------------------------------------------------
1 -> Create/activate a venv
2 -> Insert file to be processed inside folder input_data (root)
2 -> Set variables "file_path" and "n_limit" in main.py to run the process
        file_path (string) --> path + file to be processed. I have created a folder called input_data to deposit files
        n_limit (int) --> names to be processed
3 -> Executes "main.py"
4 -> You will find the results in the "results" folder (root)
    - If you want to change this folder or filenames that process generates, you can change them inside "src/core/constants.py"

-------------------------------------------------------------------------------------------------------------------------
### Executes Unit Test ###

    - It needs pytest installed in venv
